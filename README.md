# Conversational-BI_Project
An AI-powered data analysis web application built using Streamlit, Pandas, and Plotly that allows users to upload CSV datasets and ask natural language business questions to generate insights, metrics, and interactive visualizations instantly.

🚀 Features
💬 Ask Questions in Natural Language
Example:
-"Total revenue"
-"Best selling product"
-"Revenue by region"
-"Sales dashboard"

📂 Upload Any CSV Dataset
-Automatically detects key columns like:
-Revenue / Sales
-Profit
-Product
-Region
-Date

📊 Dynamic Dashboard Generation
-Bar Charts (Product-wise analysis)
-Pie Charts (Region-wise distribution)
-Line Charts (Trend over time)

🧠 AI Insights Engine
-Top performing product
-Best region
-Profitability status
-Business trends

⚡ Real-Time Analysis
-Instant responses
-Interactive visualizations

🛠️ Tech Stack
-Frontend/UI: Streamlit
-Data Processing: Pandas
-Visualization: Plotly
-Language: Python

📌 How It Works
-Upload a CSV file
-The system detects important columns automatically
-Ask business-related questions
-AI processes the query and:
-Returns answers
-Generates charts (if required)
-Provides insights

📸 Example Questions
-What is total revenue?
-Which product has highest sales?
-Show revenue by region
-Generate sales dashboard
-Show trend over time

⚠️ Limitations
-Works best when dataset contains common business columns like:
   >Revenue / Sales
   >Product
   >Region
   >Profit
-Complex or very vague questions may not always return accurate results
-Column detection is keyword-based (not full NLP yet)
-Not a fully trained AI model (rule-based logic)

🔮 Future Improvements
-Integration with LLMs (like GPT) for better understanding
-Support for more complex queries
-Better column detection using NLP
-Multi-file comparison
-Export reports (PDF/Excel)

💡 Use Cases
-Business analytics dashboards
-Student projects / hackathons
-Quick data exploration
-Non-technical users analyzing data

CONCLUSION : This project simplifies data analysis by combining AI + visualization, making it easy for anyone to extract insights from raw data without writing complex queries.
